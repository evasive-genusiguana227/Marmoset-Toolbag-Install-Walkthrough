## 🔍 Fixing Common Errors

If you see an error window, check this list to find a solution.

*   Missing DLL Files: This happens when the Visual C++ Redistributable is outdated. Navigate to the Microsoft support website and download the latest Visual Studio 2015, 2017, and 2019 redistributable. Install both the x86 and x64 versions.
*   Access Denied Errors: Antivirus software often blocks new installations. Temporarily turn off your antivirus protection during the setup process. Remember to turn it back on when the install finishes.
*   Black Screen at Startup: This occurs when the graphics card does not recognize the software. Right-click your desktop and open your Display Settings. Ensure your monitor connects to the discrete graphics card and not the integrated motherboard port.
*   Setup Failed: Clean your temporary files. Press the Windows key + R. Type %temp% and hit Enter. Select all files in this folder and delete them. Try the installer again.
